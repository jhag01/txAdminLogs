# txAdminLogs

A lightweight, standalone logging system for FiveM that routes txAdmin events to Discord.

📖 **Full documentation:** https://jhag01.github.io/fivem-docs/txadminlogs/

## Requirements

- Standalone — no framework requirements (ESX, QBCore, or ox are not needed)
- txAdmin (bundled with FXServer)

## Quick start

1. Clone into `resources/`
2. Set your Discord webhook(s) in `settings.lua`
3. `ensure txAdminLogs` in `server.cfg`
4. See the [docs](https://jhag01.github.io/fivem-docs/txadminlogs/) for webhook routing, every loggable event, and everything else.
